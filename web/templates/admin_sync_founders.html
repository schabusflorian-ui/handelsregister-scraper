How to Sync

Step 1: Run stealth scraper locally

python -m scheduler.jobs.stealth_founder_job

Step 2: Export founders to JSON

python scripts/export_founders.py --output founders.json

Or for base64 (easier to paste):

python scripts/export_founders.py --base64

Step 3: Copy and paste

Copy the output and paste it in the text area on the left.

  • Existing founders are updated with new data
  • Null values in update don't overwrite existing data
