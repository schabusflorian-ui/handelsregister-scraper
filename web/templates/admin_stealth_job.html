How It Works

  1. DuckDuckGo Search: Searches for LinkedIn profiles matching stealth founder patterns (e.g., "stealth founder germany", "building something new berlin")
  2. URL Extraction: Extracts LinkedIn profile URLs from search results
  3. Profile Scraping: Fetches public LinkedIn profile data using cloudscraper
  4. Signal Detection: Analyzes profiles for stealth signals (Specter/Harmonic-inspired):
    • Stealth keywords ("stealth mode", "building something new", etc.)
    • Career transitions ("ex-Google", "just left", etc.)
    • High-value backgrounds (FAANG, unicorns, top VCs)
    • Urgency signals ("hiring", "raising", "beta")
  5. Confidence Scoring: Calculates confidence score (0-100%) based on signals
  6. Storage: Stores qualified founders (confidence >= 30%) in database
Note: Cloud IPs may get blocked more often. The job uses conservative delays (20-60s for search, 10-30s for profiles) to minimize blocks.
